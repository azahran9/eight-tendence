# Eight-endance

A simple, private attendance tracker built for GitHub Pages.  
Tracks how many days you come to the office each month with an 8-day target.

## Features
- One-click "Check In Today"
- Add past attendance dates
- Add holidays/vacations (reduce required days)
- Auto month tracking
- Progress bar (attendance vs requirement)
- Attendance list
- Export monthly summary as text/PDF
- Fully static (no backend)
- Works on GitHub Pages instantly
